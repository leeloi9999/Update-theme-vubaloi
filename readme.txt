== Installation ==

Sau khi đã sửa xong theme muốn update theme lên web thì làm các bước sau:

Bước 1: Vào mục "giao diện" (appreance) trong trang quản trị của website (thongtin.club/wp-admin)
Bước 2: Chọn mục "themes"-> sau đó là chọn "thêm mới" -> sau đó chọn "Tải giao diện lên" -> chọn file nén dưới dạng zip của file theme của mình -> Như vậy là update theme xong
